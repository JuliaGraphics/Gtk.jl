Gtk.jl

Julia Bindings for Gtk.

Introduction

Gtk.jl is a is a Julia package providing bindings for the Gtk library: https://www.gtk.org/

Complete Gtk documentation is available at https://developer.gnome.org/gtk/stable

Installation

Install Gtk.jl within Julia using

using Pkg
Pkg.add("Gtk")

If this produces errors, please see installation troubleshooting.

Precompilation

Gtk is precompilable by normal mechanisms. For further reduction of startup time for applications that use Gtk, one can even build it into your local installation of julia.

Common Issues

If you are running Gtk on MacOS Mojave, it's likely that when you try one of te examples, a blank window will render. In order to fix this, you can run the following commands which will downgrade your version of glib to the compatible version for Gtk.jl

Command to run: 

using Homebrew
Homebrew.brew(`unlink glib`)
Homebrew.brew(`install https://raw.githubusercontent.com/Homebrew/homebrew-core/b27a055812fe620e0d3dbe67f2a424ed3a846ecf/Formula/glib.rb`)
